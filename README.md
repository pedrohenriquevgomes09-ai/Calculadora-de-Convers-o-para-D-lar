#  Calculadora de Conversão para Dólar

Este projeto é uma aplicação simples que calcula quantos dólares (USD) podem ser comprados com um determinado valor em moeda local (por exemplo, reais), com base na cotação atual ou informada pelo usuário.

## 📌 Funcionalidades

* Inserir o valor disponível na carteira
* Informar a cotação do dólar (ou usar uma padrão)
* Calcular automaticamente quantos dólares podem ser comprados
* Exibir o resultado de forma clara

## 🧮 Fórmula utilizada

dólares = valor_em_reais / cotação_do_dólar


## 🚀 Como usar

1. Clone este repositório:

git clone https://github.com/seu-usuario/calculadora-dolar.git


3. Execute o programa (exemplo em Python):

python main.py


4. Insira:

   * O valor que você tem na carteira
   * A cotação atual do dólar

5. Veja o resultado no terminal 🎉

## 💻 Exemplo de uso


Digite o valor em reais: 1000
Digite a cotação do dólar: 5.00

Você pode comprar: 200.00 USD
